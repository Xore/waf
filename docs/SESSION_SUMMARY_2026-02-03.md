## üìã Documentation Templates

### PowerShell Help Template

Use this template for all documentation work:

```powershell
<#
.SYNOPSIS
    [Short one-line description]

.DESCRIPTION
    [Detailed multi-line description of functionality,
    integration points, and what the script does]

.PARAMETER ParameterName
    [Description if script has parameters]

.EXAMPLE
    .\\ScriptName.ps1
    [Example usage]

.NOTES
    Frequency: [Schedule]
    Runtime: [Expected duration]
    Timeout: [Maximum time]
    Context: [SYSTEM or USER]
    
    Fields Updated:
    - fieldName (Type: Text/Integer/Checkbox/DateTime/WYSIWYG/Decimal)
    - fieldName (Type: ...)
    
    Dependencies:
    - [Any script dependencies]
    
    Framework Version: 4.0
    Last Updated: February 4, 2026
#>
```

### Quality Checklist Per Script

- [ ] .SYNOPSIS present and descriptive
- [ ] .DESCRIPTION detailed and explains functionality
- [ ] .PARAMETER for all parameters (if applicable)
- [ ] .EXAMPLE with usage (if applicable)
- [ ] .NOTES section complete with all metadata
- [ ] All fields documented with types
- [ ] Framework Version: 4.0
- [ ] Last Updated: Current date
- [ ] No typos or formatting errors
- [ ] Code functionality unchanged
